color-contrast()

Discouraged
Groups: CSS > Color types

This feature is discouraged. Consider using contrast-color() instead. For the rationale, see:

  • https://github.com/w3c/csswg-drafts/issues/7557
  • https://github.com/w3c/csswg-drafts/commit/36594e9e8496a5ca83f1f1ee2b4a83bcdf28753b

The color-contrast() CSS function picks a color from a list that has the most contrast with another color.
