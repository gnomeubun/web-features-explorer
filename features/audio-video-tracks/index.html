Audio and video tracks

Limited availability
Tags: api Groups: HTML > Media elements

The audioTracks and videoTracks APIs for media elements switch audio and video tracks during playback.

Learning resources on MDN

  • HTMLMediaElement: audioTracks property
  • HTMLMediaElement: videoTracks property
