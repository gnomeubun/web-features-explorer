toGMTString()

Discouraged
Tags: javascript

This feature is discouraged. For the rationale, see:

  • https://tc39.es/ecma262/multipage/additional-ecmascript-features-for-web-browsers.html#sec-additional-ecmascript-features-for-web-browsers

The toGMTString() method of Date objects is an alias to the toUTCString() method.
