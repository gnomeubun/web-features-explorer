Mutation events

Discouraged
Tags: api Groups: DOM

This feature is discouraged. Consider using MutationObserver instead. For the rationale, see:

  • https://www.w3.org/TR/2024/WD-uievents-20240906/#legacy-mutationevent-events

Mutation events like DOMSubtreeModified, DOMNodeInserted, or DOMNodeRemoved fire when DOM changes occur.
